¿Por qué es tan GENIAL?

CIT es un equipo que destaca dentro del resto de equipos de la UNITEC debido a que es el único orientado al Desarrollo Técnologico, y a su gran trabajo en equipo de sus integrantes con el objetivo fundamental de ofrecer soluciones tecnologicas para la sociedad

Misión

Desarrollar soluciones integrales basadas en medios tecnológicos, innovadores, ecológicos y de bajo impacto económico que solventen problematicas de la sociedad.

Visión

Ser reconocido por la sociedad como el equipo referente de P.I. de la UNITEC para las búsqueda y solución de problemas basados en texcnologías con una buena relación costo-servicio.

Objetivo General

Conformar un equipo multidisciplinario, que basado en las ultimas tecnologias pueda diseñar, crear e implementar soluciones en el área técnica apoyados en la innovación, de forma ecológica y con el menor costo posible a ser aplicados a las sociedad.

Mejores Herramientas!

En nuestro equipo tenemos gente con amplia experiencia en diversas Herramientas informáticas

JavaScript

Es un lenguaje de programación orientado a objetos débilmente tipado que se utiliza principalmente para la creación páginas web dinámicas.

Angular

Es un framework para aplicaciones web desarrollado en TypeScript, se utiliza para crear y mantener aplicaciones web de una sola página.

Laravel

Laravel es un framework de código abierto para desarrollar aplicaciones y servicios web con PHP 5 y PHP 7.

Bootstrap

Es el framework más popular de CSS y JS para el diseño y creación de sitios y aplicaciones web.

PHP (preprocesador de hipertexto)

Es un lenguaje de programación backend incrustado en HTML utilizado para crear páginas web dinámicas.

Wordpress

Es un sistema de gestión de contenidos enfocado a la creación y mantenimiento de cualquier tipo de página web.

Adobe (Herramientas)

Es un conjunto de Herramientas indispensables para el Diseño grafico y/o digital que permiten crean recursos audiovisuales.
